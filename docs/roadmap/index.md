# 🗺️ OpenClaw CN 发展路线图

> **我们的愿景**：打造更安全、更懂中文环境、适配 DeepSeek 等国产大模型的 Local Agent 基础设施。

OpenClaw CN 刚刚起步。我们拒绝画大饼，我们只做看得见、摸得着的迭代。以下是我们 2026 年上半年的核心作战计划。

## 🚀 第一阶段：筑基 (2026 Q1 - 2月)
**目标：跑通流程，扫清障碍，让 DeepSeek 能在 OpenClaw 上“丝滑”运行。**

- [x] **基础设施搭建**
    - [x] 完成 Gitee/GitHub 双向同步管道
    - [x] 上线基于 VitePress 的中文官网
    - [x] 部署国内 CDN 加速与域名备案体系
- [x] **文档本土化**
    - [x] 核心文档《快速开始》中文化与环境适配修正（目前主推源码级稳妥安装）
    - [x] 上线 DeepSeek 专属配置指南
- [x] **国产模型适配 (里程碑)**
    - [x] **DeepSeek 源码级原生支持**：实现 `provider: deepseek`，无需代理配置。
    - [x] **DeepSeek R1/V3 专项适配**：完成 Onboard 向导集成。

## 🔗 第二阶段：连接与互助 (2026 Q1 - 3月)
**目标：彻底降低上手门槛，建立互助社区，连接中国特有的互联网服务。**

- [ ] **运营与极简部署生态**
    - [ ] **全平台脚本统一**：针对不同系统环境，推出行为一致的安装、运维、升级、卸载一键脚本。
    - [ ] **保姆级视频教程**：录制并发布配套的实操视频，覆盖从环境配置到 Agent 唤醒的全流程。
- [ ] **社群与“小龙虾”互助生态**
    - [ ] 筹备并开启系列直播课程，手把手带领小白跨越技术鸿沟。
    - [ ] 建立“小龙虾互助生态”，让每一位社区伙伴都能快速安装体验，形成良好的“传帮带”开源氛围。
- [ ] **本地化插件生态 (Plugins)**
    - [ ] **企业微信 (WeCom) 连接器**：支持通过企业微信接收指令与推送消息，打通企业级办公场景。
    - [ ] **中文搜索增强**：集成国内主流网页搜索 API，大幅优化中文语境下的检索与信息抓取能力。

## 🌳 第三阶段：繁荣中国生态 (2026 Q2 - 4月至5月)
**目标：沉淀优质资产，打造最懂中国开发者的 Skills 生态圈与杀手级应用。**

- [ ] **打造中国 Skills 本地生态**
    - [ ] 全面梳理并引入原版 OpenClaw 中优质的 Skill 资产。
    - [ ] 结合国内实际业务与互联网环境进行深度本地化改造，构建纯正的中国 Skills 插件库。
- [ ] **Killer Apps (杀手级场景 Demo 落地)**
    - [ ] **全网聚合跑腿助手 (Gig Worker Agent)**：统一监听多平台订单，通过企业微信/飞书极速推送。
    - [ ] **售后服务闭环系统 (Service Loop Agent)**：自动化填表与工单流转，彻底解放维修师傅的双手。
