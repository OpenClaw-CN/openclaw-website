# 🧩 插件市场 (Plugin Market)

OpenClaw 不仅仅是一个聊天机器人，它是一个拥有“手脚”的超级员工。
通过加载不同的 **Skills (技能)**，它可以操作你的浏览器、终端，甚至连接你的企业软件。

## 🔥 核心官方插件 (Core Skills)

这些能力已经内置在 OpenClaw 中，开箱即用，无需安装。

| 插件图标 | 插件名称 | 功能描述 | 权限等级 |
| :---: | :--- | :--- | :--- |
| 💻 | **Terminal Pro** | **终端指挥官**<br>允许 Agent 执行 Shell/CMD 命令。支持软件安装、Git 操作、系统维护。 | ⭐⭐⭐ (高危) |
| 📂 | **File Manager** | **文件管家**<br>全能的文件读写能力。支持遍历目录、创建文件、批量重命名、读取 PDF/TXT 内容。 | ⭐⭐ |
| 🌐 | **Web Browser** | **全网通**<br>集成 Puppeteer/Browser-Use。可以打开真实的浏览器窗口，抓取网页数据、点击按钮。 | ⭐⭐ |

## 🚧 孵化中 (Coming Soon)

中国社区正在全力开发针对本土办公场景的超级插件，对应 **Roadmap 第二阶段** 计划：

::: info 🚀 正在开发
* **飞书 (Feishu) 连接器**：让 AI 自动读取飞书文档、发送群消息、更新多维表格。
* **企业微信 (WeCom) 助手**：接入企微 API，实现客户群自动回复与商机捕捉。
* **中文搜索增强**：集成必应/百度搜索 API，替代无法访问的 Google Search。
:::
