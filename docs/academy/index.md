## 什么是「赛博养虾」？

<div class="academy-section-intro">

<p>在 OpenClaw 中国社区，我们亲切地将部署和开发 Local Agent（本地智能体）的过程称为<strong>「养虾」</strong>。</p>

<p>传统的 AI 只是个困在网页对话框里的打字机（你输入，它回答），而 OpenClaw 是一只真正拥有心跳、能记住你、还能替你干活的「赛博大龙虾」。</p>

<p class="academy-section-intro__steps"><strong>成为一名顶级饲养员，你需要掌握以下五个进阶步骤：</strong></p>

</div>

<div class="academy-feature-grid academy-feature-grid--steps">

<div class="academy-feature-item">🌊 <strong>建虾池（本地部署）</strong>：搞定本地运行环境。所有数据和算力都在你的机器上闭环跑，不经过任何第三方，为你的专属 Agent 打造绝对隐私、安全无监听的「恒温水质」。</div>

<div class="academy-feature-item">🧬 <strong>注虾魂（记忆系统）</strong>：配置 <code>SOUL.md</code> 和 <code>MEMORY.md</code>。告别「聊完就忘」的健忘症，为电子虾注入长期记忆和专属人设。它不仅能记住工作流，还能真正记住「你是谁」。</div>

<div class="academy-feature-item">🦾 <strong>装铁钳（技能与工具）</strong>：为龙虾安装专属 Skills（技能插件）。就像给手机装 App 一样，给它装上能读写文件、操作浏览器、调用外部 API 的「机械双钳」，让它拥有改变物理世界的能力。</div>

<div class="academy-feature-item">🐙 <strong>伸触角（多渠道通信）</strong>：打破「输入框」的牢笼！给龙虾接入飞书、钉钉…你习惯用什么社交工具，龙虾的触角就延伸到哪里，随时随地听你召唤。</div>

<div class="academy-feature-item">⚡ <strong>唤心跳（心跳机制）</strong>：激活 Agent 的 Heartbeat。它不再是戳一下才动一下的死物，而是拥有「生物钟」的数字生命。它会定期自动醒来，巡视任务、主动向你汇报，成为 24 小时在线的完美私助。</div>

</div>
